# DIY Mini Retro Arcade (Bartop Console)

This project is a DIY mini retro arcade cabinet in bartop form factor.  
It is built from scratch using MDF panels and off-the-shelf electronics.

The goal is to design and build a fully functional standalone arcade machine
that runs classic retro games using emulation software.

---

## Project Overview

- Bartop-style wooden arcade cabinet  
- Integrated display, speakers, and arcade controls  
- Runs retro games (NES, SNES, MAME, PSX)  
- Android-based system (Android TV Box)  
- Built as a learning project focused on electronics and woodworking
