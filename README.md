# DIY Mini Retro Arcade (Bartop Console)

This project is a DIY mini retro arcade cabinet inspired by classic bartop arcade machines.

## Goal
Build a compact standalone arcade console with a custom wooden cabinet, arcade controls, and an Android/Linux-based system capable of running classic 2D retro games (NES, SNES, MAME, PSX).

## Status
Project is in planning and early build stage.

## Hardware (planned)
- Arcade joystick and buttons
- LCD monitor
- Android/Linux device
- Wooden cabinet
- Power supply and wiring

More updates will be added during the build.
