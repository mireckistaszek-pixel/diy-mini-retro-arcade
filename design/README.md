# Design Files

This folder contains design materials for the DIY Mini Retro Arcade project.

## Cabinet Design
- Bartop-style cabinet made from 12mm MDF
- Compact tabletop size
- Front-mounted arcade controls
- Rear ventilation with 2x 80mm fans

## Control Layout
- 1 joystick
- 8 action buttons
- Start / Select buttons

## Electronics Layout
- Android TV Box mounted inside the cabinet
- Internal 14" LCD monitor
- USB Zero Delay encoder
- Stereo speakers mounted behind front grill
